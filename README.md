# my-market-app

## **Курс Yandex practicum. Middle Java-developer. Sprint6**

**Данное веб-приложение «Витрина интернет-магазина» является модификацией приложения спринта №5, переписанное с использованием Spring Boot на реактивном стеке технологий.**

- Сборка приложения осуществляется с помощью систем сборки Maven.  
- Код приложения написан на Java 21  
- База данных приложения H2
- Приложение использует Spring Data R2DBC
- Приложение упаковано в Executable JAR и запускается в Embedded Web Server (Netty)
- Использованы готовые *.html шаблоны  

Веб-приложение представляет собой витрину товаров, которые пользователь может положить в корзину и купить.
Приложение состоит из шести основных частей (модулей): страницы витрины товаров, доступных для просмотра и покупки; 
страницы товара; страницы корзины покупателя; страницы всех заказов; страницы заказа; сервиса покупки (не имеет отдельной веб-страницы)

- Добавлен веб-интерфейс добавления товара  
- Приложение можно собрать из исходников и запустить локально  
- Добавлен Dockerfile для развёртывания и запуска веб-приложения в Docker-контейнере  


#### **Команды для работы в Docker:**
Сборка и запуск:  
_**docker-compose up --build**_  

Запустить в фоне:  
_**docker-compose up -d**_ 

Остановить:  
_**docker-compose down**_
