# my-market-app

## **Курс Yandex practicum. Middle Java-developer. Sprint8**

**Данное веб-приложение «Витрина интернет-магазина» является развитием приложения спринта №7, c добавлением Sprting Security и Keycloack**

Изменения в текущей версии:
- Добавлена авторизация покупателя в приложении «Витрина интернет-магазина»;
- Добавлена авторизация взаимодействия между приложением «Витрина интернет-магазина» и RESTful-сервисом платежей;
- Добавлен Spring Security в качестве модуля авторизации Spring Boot-приложений;
- Для логина/логаута пользователя используются дефолтные настройки Sprint Security;
- Данные пользователей заранее сохранены в БД;
- С каждым авторизированным пользователем связана только своя корзина;
- Неавторизованные пользователи имеют доступ на страницу с товарами, но не могут заполнять корзину. При попытке заполения корзины им предлагается перейти на страницу авторизации;
- Пользователь с ролью admin, имеет доступ к панели добавления товаров;
- Добавлен сервер авторизации Keycloack, настроенный для работы в Docker контейнере;
- Модифицированы тесты для работы со Spring Security;
- Модифицирована страница товаров, для отображения текущего пользователя и добавлена кнопка смены пользователя.

Изменения в версии спринта №7:
- Добавлен RESTful-сервис платежей **payment-service** с использованием Spring WebFlux
- Добавлен сервер **Redis** для организации кеширования
- Написана OpenAPI спецификация интеграции основного приложения и сервиса платежей

Общая информация о проекте:
  
- Сборка приложения осуществляется с помощью систем сборки Maven
- Код приложения написан на Java 21 
- База данных приложения H2
- Приложение использует Spring Data R2DBC
- Приложение упаковано в Executable JAR и запускается в Embedded Web Server (Netty)
- Использованы готовые *.html шаблоны


Веб-приложение представляет собой витрину товаров, которые пользователь может положить в корзину и купить.
Приложение состоит из шести основных частей (модулей): страницы витрины товаров, доступных для просмотра и покупки; 
страницы товара; страницы корзины покупателя; страницы всех заказов; страницы заказа; сервиса покупки (не имеет отдельной веб-страницы)

- Добавлен веб-интерфейс добавления товара  
- Приложение можно собрать из исходников и запустить локально  
- Добавлен Dockerfile для развёртывания и запуска веб-приложения в Docker-контейнере  


#### **Команды для работы с my-market-app в Docker:**
Сборка и запуск:  
_**docker-compose up --build**_  

Запустить в фоне:  
_**docker-compose up -d**_ 

Остановить:  
_**docker-compose down**_  


#### **Команда для работы с Keycloack в Docker:**
Сборка и запуск:  
_**docker compose -f docker-compose.keycloack.yml up -d**_
